## How To Use

### Installation

1. To run a project locally, first clone the repo

   ```sh
   git clone https://github.com/The-TRAE-project/Trae_front.git
   ```

2. Go to the project directory

   ```sh
   cd Trae_front
   ```

3. Install all packages

   ```sh
   npm install
   ```

4. Run the project

   - In Development Mode:

   ```sh
   npm run dev
   ```

   - In Production Mode:

   ```sh
   npm run build
   ```

   ```sh
   npm run preview
   ```
